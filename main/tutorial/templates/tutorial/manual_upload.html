How to upload manually

Use the following JSON files as reference if you want to upload data without the Butler module: https://github.com/Hartvi/butler/tree/main/example_upload_dictionaries

When uploading, the following steps are recommended to be taken:
1. In the JSON you want to upload, find any absolute references to files, e.g. /abs/path/to/file.png
2. Replace absolute paths with the base file names. E.g. "/abs/path/to/file.png" => "file.png"
3. The file bytes are loaded from the absolute file paths and added to the request's request.files attribute with the base name as the key, i.e. {"file.png": open("abs/path/to/file.png")}
4. The uploading happens. See here for the tutorial on how the uploading module works.
